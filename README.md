# My Admission Guide

Independent educational consultancy platform helping students navigate admission processes across India's top engineering and professional institutions.

## Website

**Domain:** [myadmissionguide.in](https://myadmissionguide.in)

## Overview

A static HTML website providing:
- Admission guidance for top engineering colleges in Bangalore/India
- College comparison and detailed institution pages
- College Predictor tool (JEE Main rank-based)
- Counseling support and eligibility information

## Pages

- **Homepage** — Overview, featured colleges, testimonials
- **College Pages** — RV College, Ramaiah, BMS, Dayananda Sagar, Nitte, Sir MVIT, BIT, CMRIT, REVA, Alliance
- **College Predictor** — Rank-based college prediction tool using JEE Main cutoff data
- **About Us** — Company information and services
- **Contact Us** — Contact form and office details
- **Legal** — Privacy Policy, Terms & Conditions, Disclaimer

## Tech Stack

- Pure HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Inter, Poppins)
- No build tools or frameworks — static site

## SEO

- `robots.txt` — allows all crawlers
- `sitemap.xml` — all pages listed with priorities
- Google Search Console verified via meta tag
